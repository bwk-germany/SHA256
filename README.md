# SHA256

## This is the SHA256 github repository of BWK Germany.

The integrity of a file can be verified using the widely used SHA256 algorithm.

SHA stands for Secure Hash Algorithm.
The 256 represents 256 bits. 

Eight bits make up one byte. That is, 256 bits correspond to 32 bytes.

In decimal notation, one byte is a number between 0 and 255. In hexadecimal notation (where 9 is followed by a through f), a byte can be represented by two characters (0-9, a-f).

For this reason, the hash value for 32 bytes is 64 characters long, and the SHA256 output looks like this, for example:

8d23cf6c86e834a7aa6eded54c26ce2bb2e74903538c61bdd5d2197997ab2f72
